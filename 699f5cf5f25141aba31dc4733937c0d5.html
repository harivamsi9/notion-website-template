Description
• Developed a distributed cloud-based networked file storage service (SurfStore) based on Dropbox which syncs files to and from the “cloud” where client and server interactions are done with gRPC and implemented TCP/IP connections to listen and serve multiple clients with go routines and created virtual servers mappings. • Created blocks, hashes, hashlists for each file in a server, and file versions are used to maintain consistency of data upon modification/deletion across multiple clients. • Implemented using consistent hashing to scale the basic SurfStore to support 10’s thousands of clients and used log replication part of the RAFT protocol for fault tolerance. Built using Go, gRPC, Consistent Hashing, RAFT, SQL.
